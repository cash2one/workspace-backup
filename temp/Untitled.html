网站数据接口


全局接口


  • 赛后讨论点赞接口
    • 地址：/api/like
    • 方法：GET
    • 参数：
      • 比赛ID - Int
      • 球队ID - Int
    • 示例：http://sports.baofeng.com/api/like/1/2
    • 响应：
  • {
  •     status : 1,
  •     num : 23
  • }
    • 说明：
      • status - 状态
        • 值：0 - 失败，1 - 成功
      • num - 该球队点赞后的总数
  • 根据赛事ID返回该赛事下的新闻
    • 地址：/api/getEventNews
    • 方法：GET
    • 参数：
      • 赛事ID - Int
      • 数量 - Int
    • 示例：http://sports.baofeng.com/api/getEventNews/1/18
    • 响应：
  • {
  •   status : "1",
  •   news : [
  •     {
  •       id: "1",
  •       title: "我是一个标题",
  •       type: "literal",
  •       site: "腾讯体育",
  •       image: "http://image.sports.baofeng.com/2323",
  •       play_url: "http://sports.qq.com/xxxx.html",
  •       created_at: "03-15 18:03"
  •     },
  •     ...
  •   ]
  • }
    • 说明：
      • status - 状态
        • 值：0 - 失败，1 - 成功
      • news - 新闻列表
        • id - 新闻ID
        • title - 新闻标题
        • type - 新闻类型（literal - 文字，video - 视频）
        • site - 新闻来源
        • image - 新闻图片
        • play_url - 视频播放地址
        • created_at - 创建时间（时间是后端处理过的样式）
  • 根据赛事ID返回该赛事下的视频
    • 地址：/api/getVideosByEvent
    • 方法：GET
    • 参数：
      • 赛事ID - Int
      • 数量 - Int
    • 示例：http://sports.baofeng.com/api/getVideosByEvent/1/16
    • 响应：
  • {
  •   status : "1",
  •   videos : [
  •     {
  •       id: "2",
  •       match_id: "1",
  •       title: "hhahah two",
  •       type: "highlight",
  •       site: "",
  •       duration: "0",
  •       image: "",
  •       play_url: "",
  •       play_html: null,
  •       created_at: "2016-03-15 18:06:53",
  •       updated_at: "2016-03-15 18:06:53"
  •     },
  •     ...
  •   ]
  • }
    • 说明：
      • status - 状态
        • 值：0 - 失败，1 - 成功
      • videos - 视频列表
        • id - 自增 ID
        • match_id - 比赛 ID
        • type - 该视频类型（录像（reply）、集锦（highlight））
        • title - 视频标题
        • site - 来源站点
        • image - 封面图
        • play_url - 播放页地址
        • play_html - 由人工填充的播放使用的 HTML 代码片断
        • duration - 视频时长
        • created_at - 创建时间
        • updated_at - 更新时间

首页接口


  • 首页专栏推荐
    • 地址： /api/getSection
    • 方法：GET
    • 参数：
      • 页码 - int
      • 条数 - int
    • 示例： http://sports.baofeng.com/api/getSection/1/6
    • 响应：
  • {
  • "status": 1,
  • "posts": [
  •     {
  •         "id": "95",
  •         "section_id": "11",
  •         "title": "巨星今何在：1996欧洲杯捷克黄金一代",
  •         "site": "bfonline",
  •         "publish_tm": "2016-07-06 13:00:35",
  •         "section_logo": "http://image.sports.baofeng.com/c2fedafe5cf3d295f2d2efc447f7b1b5",
  •         "section_name": "说球",
  •         "url": "http://sports.baofeng.com/section/detail/95"
  •     },
  •     ...
  • }
    • 字段说明
      • status
        • 值：0 - 失败，1 - 成功
      • posts
        • id - 专栏新闻ID
        • section_id - 专栏ID
        • title - 专栏新闻标题
        • site - 专栏新闻来源
        • publish_tm - 发布时间
        • section_logo - 专栏图标
        • section_name - 专栏明称
        • url - 专栏新闻详情页地址
    • 首页节目推荐
    • 地址： /api/getProgram
    • 方法：GET
    • 参数：
      • 页码 - int
      • 条数 - int
    • 示例： http://sports.baofeng.com/api/getProgram/1/3
    • 响应：
  • {
  • "status": 1,
  • "posts": [
  •     {
  •         "id": "283",
  •         "image": "http://image.sports.baofeng.com/b20cdf6ff1ac288210f1690115ae5251",
  •         "play_url": "http://sports.sina.com.cn/euro2016/video/footballtactics/22/#250660469",
  •         "title": "《欧洲杯道中道》第22期：最强东道主期待最强决战",
  •         "site": "sina",
  •         "program_name": "欧洲杯道中道"
  •     },
  •     ...
  • }
    • 字段说明
      • status
        • 值：0 - 失败，1 - 成功
      • posts
        • id - 节目视频ID
        • image - 节目视频封面
        • title - 节目视频标题
        • site - 节目视频来源
        • play_url - 节目视频播放地址
        • program_name - 节目名称
